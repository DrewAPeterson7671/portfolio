How I Got Started

My enthusiasm to code started in high school. I remember pulling up to a stop light in my car and thinking "If weight sensor equals '> 500lbs', then...". Learning code changed how I thought about everything.

In the mid-2000s, I played a sports simulation game (Fast Break Basketball!) that is a spreadsheet sports game. I loved it so much, I had to run my own league, my way! I made the webpages in Dreamweaver and found an inexpensive hosting service. It thrived! But I wanted to take it further, correct everything wrong with it. And make it web based! Why not? So I made some HTML pages ("What are all these .css files cluttering everything up?" "What are these 'div' tags all over this example page? They don't do anything... "). So to make a league, I have to make forms to create leagues and teams and players. No Problem. ("Huh... When I hit 'submit', it isn't... retaining the data..." There must be more going on under the surface, something I haven't seen."). Then I downloaded MySQL (Soooo...... How does this form connect to it...).
